# Predictive Modeling of Type 2 Diabetes Risk in U.S. Adults

> A machine learning approach to identify risk factors for Type 2 diabetes using large-scale U.S. public health data.

## 📌 Overview
This project develops predictive models to identify risk factors for Type 2 diabetes in U.S. adults using the CDC Behavioral Risk Factor Surveillance System (BRFSS) 2021 dataset. The analysis evaluates multiple predictive modeling approaches and highlights key demographic, health, and socioeconomic factors associated with diabetes risk.

## 🎯 Objectives
- Develop predictive models for Type 2 diabetes risk
- Compare performance across multiple modeling approaches
- Identify key demographic, health, and socioeconomic risk factors
- Support early detection and public health decision-making

## 📊 Data Source
- CDC Behavioral Risk Factor Surveillance System (BRFSS) 2021 dataset
- Large-scale U.S. population-based health survey data

## ⚙️ Methodology
- Data preprocessing and cleaning
- Feature selection
- Handling class imbalance
- Model development and performance evaluation

### Models Implemented
- Logistic Regression
- Decision Tree
- Neural Network

## 📈 Results
- The neural network achieved the best performance:
  - **AUC:** 0.811
  - **Accuracy:** 74%
- Logistic regression and decision tree models provided useful baseline comparisons
- Important risk factors included:
  - Age
  - Body Mass Index (BMI)
  - Socioeconomic conditions

## 🛠️ Tools & Technologies
- SAS Enterprise Miner
- Statistical modeling and machine learning methods

## ⚠️ Limitations
- This is an observational analysis and does not establish causality
- The BRFSS dataset is based on self-reported survey responses
- Class imbalance may affect predictive performance
